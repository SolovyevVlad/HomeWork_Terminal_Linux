Linux terminal (GitBash) commands

1) Посмотреть где я ---->	pwd
2) Создать папку---->	mkdir folder (папка с названием "folder")
3) Зайти в папку---->	cd folder/ 
4) Создать 3 папки---->	mkdir folder_1 folder_2 folder_3
5) Зайти в любоую папку---->	cd folder_2
6) Создать 5 файлов (3 txt, 2 json)---->	touch file_1.txt file_2.txt file_3.txt file_4.json file_5.json
7) Создать 3 папки---->	mkdir inner_folder_1 inner_folder_2 inner_folder_3
8. Вывести список содержимого папки---->	ls -la
9) + Открыть любой txt файл---->	vim file_1.txt
10) + написать туда что-нибудь, любой текст.---->	Нажать "i". Ввести текст.
11) + сохранить и выйти.---->	Нажать "Esc". Ввести ":wq", нажать "Enter".
12) Выйти из папки на уровень выше---->		cd..
13) переместить любые 2 файла, которые вы создали, в любую другую папку.---->	mv folder_2/file_1.txt folder_2/file_2.txt folder_1
14) скопировать любые 2 файла, которые вы создали, в любую другую папку.---->	cp folder_1/file_1.txt folder_2/file_4.json folder_3
15) Найти файл по имени---->	find . -iname file_4* 		("." - ведется поиск в текущей директории. Так же можно указать путь, где должен происходить поиск файла по имени)
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает.---->	tail -f folder_1/file_1.txt (указан путь к файлу, так как следуя по заданию из пункта 12, мы сейчас находимся в директории с папками без файлов)
17) вывести несколько первых строк из текстового файла---->	head -5 folder_1/file_1.txt 
18) вывести несколько последних строк из текстового файла---->	tail -5 folder_1/file_1.txt 
19) просмотреть содержимое длинного файла (команда less) изучите как она работает.---->		less file_3.txt
20) вывести дату и время---->	date
